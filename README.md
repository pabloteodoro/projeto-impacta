# 🎓 Projeto Impacta - Portal do Aluno

Este projeto é uma recriação moderna de um **Portal do Aluno**, inspirado em sistemas acadêmicos reais, desenvolvido com foco em **boas práticas de desenvolvimento Full Stack**.

A aplicação possui autenticação, dashboard interativo, componentes responsivos e integração com banco de dados.

---

# 🚀 Tecnologias Utilizadas

### Frontend
- **Next.js 16**
- **React**
- **TypeScript**
- **TailwindCSS**
- **Lucide Icons**

### Backend
- **Next.js Server Components**
- **JWT Authentication**
- **Prisma ORM**

### Banco de Dados
- **PostgreSQL**

---

# 🧩 Funcionalidades

### 🔐 Autenticação
- Login com validação
- Autenticação via **JWT**
- Sessão armazenada em **cookies**
- Redirecionamento automático para login se não autenticado

---

### 📊 Dashboard do Aluno
Interface moderna contendo:

- Avisos importantes
- Informações do aluno (nome, RA e curso)
- Atalhos acadêmicos
- Cards informativos

---

### 📅 Calendário Inteligente
Componente interativo que:

- Mostra o **mês e ano atual automaticamente**
- Gera os **dias corretamente na semana**
- Destaca **o dia atual**
- Permite **navegação entre meses**
- Possui **modal com calendário completo**

---

### 📱 Responsividade
Interface totalmente adaptável para:

- Desktop
- Tablet
- Mobile

Incluindo:

- Sidebar colapsável no mobile
- Layout fluido com TailwindCSS
